# 🚀 Cybersecurity Home Lab

## Overview
This project demonstrates a basic cybersecurity home lab environment created using VirtualBox.

## Technologies Used
- Kali Linux
- Ubuntu Server
- Apache2
- OpenSSH
- Nmap
- VirtualBox

## Features
- Hosted a custom webpage using Apache
- Enabled SSH remote access
- Performed service enumeration using Nmap
- Established communication between Kali and Ubuntu

## Open Ports Discovered
Port  Service 
 22   SSH
 80   HTTP (Apache)


## Learning Outcomes
- Linux Administration
- Networking Fundamentals
- Client-Server Architecture
- Service Enumeration
- Remote Server Management
